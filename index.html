Mau belajar apa hari ini?

HTML Tutorials

HTML merupakan singkatan dari Hypertext Markup Language. HTML digunakan untuk mendesain laman web dengan bantuan Hypertext dan Markup Language, dimana Hypertext berfungsi untuk menghubungkan antar laman web dan Markup Language merupakan bahasa untuk mendesain laman web.

CSS Tutorials

Cascading Style Sheets, atau juga disebut CSS, merupakan bahasa yang ditujukan untuk memudahkan proses pembuatan visualisasi laman web agar menjadi lebih indah. CSS memudahkan proses visualisasi dengan cara menambahkan syles ke dalam bagian-bagian laman web yang kita inginkan.

PHP Tutorials

Istilah PHP adalah akronim untuk Preprocessor Hypertext. PHP adalah Bahasa scripting sisi server yang dirancang khusus untuk web development. PHP dapat dengan mudah tergabung dalam file HTML.

Javascript Tutorials

Javascript pertama dikembangkan oleh Brendan Eich pada tahun 1995. Awalnya, bernama LiveScript tetapi kemudian diberi nama Javascript. JavaScript digunakan untuk mengatur fungsionalitas dari struktur laman web.
